Инструкция

Для появления нового абзаца просто кликайте по области ограниченной серым контуром.

Небольшое пояснение по обозначениям в тексте:
"" - прямая речь гг, либо его мысли.
() - текст на экране монитора, с которым взаимодействует гг.

Две цифры снизу - используются только для сюжетных выборов, но вы можете кликать по ним сколько угодно, если желаете.

Для того, чтобы открыть настройки, просто кликните на три точки справа сверху.

Цветовая схема - выбор цвета фона и текста.

Время - это рудимент, который планировалось использовать в повествовании, но не сложилось. Вы можете отображать его для удобства.

Автоскроллинг - это автоматический фокус на последнем появившемся абзаце, который включен по дефолту, чтобы вам не скролить собственноручно. Да, я потакаю вашей лени...

Красная кнопка - удаляет весь текст с "экрана". Вернуть его можно только перезагрузив страницу и начав занаво прокликивать весь уже пройденный текст.

Не нажимайте её бездумно!

Можете удалить данную инструкцию после прочтения, дабы она не мешала.

P.S Настоятельно рекомендую читать в полной темноте и тишине.
